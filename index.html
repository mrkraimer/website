DBRecordProviders

DBRecordProviders is a comparison of:

channel access
The epics base network protocol that has existed since the late 1980's.
ca
The ca provider that comes with pvAccessCPP. It is a client side provider that uses channel access as the network protocal.
qsrv
A server side network protocol the directly access DBRecords in a epics ioc.

bloscCodec

bloscCodec implements a database and client code to test a compress/decompress facility for numeric scalar arrays.

testDatabase

testDatabase has a database and client code to test some esoteric features for providers pva and ca connecting to DBRecords in an EPICS IOC.
